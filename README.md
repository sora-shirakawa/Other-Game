# Orbit Pulse Runner

スマートフォンでの反復プレイを意識した、1プレイ短時間のHTML5アーケードゲームです。

## 遊び方
- **開始**: 「スタート / リスタート」ボタン。
- **操作**: 左右スワイプ（PCは `←` `→`）で3レーンを移動。
- **目的**: 円軌道上を周回しながら敵を回避し、できるだけ長く生き残る。
- **パワーアップ**:
  - Shield: 一定時間無敵
  - Pulse: 回転速度アップ
  - Life: ライフ回復

## リプレイ性を高める設計
- プレイ時間に応じたウェーブ難易度上昇
- 敵出現のランダム化（レーン、進行方向、速度）
- パワーアップ出現のランダム化
- ローカルストレージでベストスコア保存

## Unity(C#)移植しやすい理由
- `Player / Enemy / Powerup / Particle` の責務を分離したデータ構造
- `update(dt)` 中心のゲームループで `Update()` への写像が容易
- 軌道位置計算が `Vector2(center) + direction * radius` でそのまま移せる
